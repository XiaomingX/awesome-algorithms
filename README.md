# awesome-algorithms（算法学习资源精选：从入门到进阶的实用指南）


## 一、可视化学习网站【新手友好，直观理解原理】
可视化是入门算法的最佳方式——通过动画拆解每一步逻辑，复杂概念（如排序、递归）瞬间变清晰。

- **[AlgoVisualizer](http://algo-visualizer.jasonpark.me/)**  
  【核心优势】支持用 Python、JS 等语言自定义代码，实时生成算法动画；不仅能看现成案例，还能动手验证自己写的逻辑。  
  【适合人群】想“边动手边理解”的初学者。

- **[VisuAlgo](http://visualgo.net)**  
  【核心优势】覆盖 80+ 算法/数据结构（排序、树、图、动态规划等），支持“分步演示”“参数调节”（如数组长度、排序速度），甚至自带习题模式，看完就能练。  
  【适合人群】需要系统梳理基础知识点的新手。

- **[USFCA 数据结构可视化](http://www.cs.usfca.edu/~galles/visualization/Algorithms.html)**  
  【核心优势】按“排序、搜索、图、树”分类清晰，动画可暂停/重播，每步都配文字解释（如“交换索引 1 和 3 的元素”），纯零基础也能跟上。  
  【适合人群】对抽象概念敏感，需要“逐帧拆解”的学习者。


## 二、入门教程网站【系统打基础，附代码实现】
看完可视化后，用这些教程巩固理论+动手写代码，形成“理解-实践”闭环。

- **[W3School 数据结构教程](https://www.w3schools.in/data-structures-tutorial/intro/)**  
  【核心优势】纯文字+极简示例，聚焦“是什么”（如数组vs链表的区别），没有复杂推导，适合完全没接触过算法的“纯小白”。  
  【适合人群】算法零基础，想先搞懂基础概念的人。

- **[Geeks for Geeks 算法基础](http://www.geeksforgeeks.org/fundamentals-of-algorithms/)**  
  【核心优势】内容极全（从“数组初始化”到“贪心算法应用”），每个知识点配 Java/Python/C++ 多语言代码，还附“面试高频考点”标注，实用性拉满。  
  【适合人群】想兼顾“基础”和“面试”的学习者。

- **[Programiz 数据结构与算法](https://www.programiz.com/dsa)**  
  【核心优势】交互式教程（可在线运行代码），章节逻辑循序渐进（先数据结构，再算法），重点用彩色标注（如“时间复杂度关键结论”），记笔记很方便。  
  【适合人群】喜欢“边学边练”，需要明确重点的开发者。


## 三、复杂度分析工具【面试必背，效率对比一目了然】
“大O复杂度”是算法的“核心评分标准”，这个工具帮你快速记牢重点。

- **[Big O 复杂度速查表](http://bigocheatsheet.com/)**  
  【核心优势】不仅有“算法时间/空间复杂度表”（如冒泡排序 O(n²)、二分查找 O(logn)），还附“数据结构操作效率对比”（如数组查改快、链表增删快），支持下载 PDF 打印，备考面试直接背。  
  【适合人群】需要快速掌握复杂度分析，应对面试提问的人。


## 四、视频教程【听不懂文字？看动画+讲解】
视频比文字更易聚焦，推荐“国内外互补”——YouTube 资源看深度，B站资源看适配性。

  【B站替代】直接搜“FreeCodeCamp 算法 中文”，有高清搬运版。

- **[Abdul Bari：算法入门](https://www.youtube.com/watch?v=0IAPZzGSbME&list=PLDN4rrl48XKpZkf03iYFl-O29szjTrs_O)**

  【B站替代】搜“Abdul Bari 算法 中文字幕”，有完整合集。

### 国内优质（适配中文语境+面试需求）
- **[B站：代码随想录](https://space.bilibili.com/525424077)**

- **[B站：正月点灯笼](https://space.bilibili.com/302417610)**  
  【核心优势】用“生活化比喻”（如“查字典讲二分”）+ 动画，讲透复杂算法（如图论、动态规划），零基础也能听懂。


## 五、在线课程【系统性进阶，配作业+证书】
想系统学理论或拿认证？选这些经过验证的经典课程。

- **[MIT 6.006：算法导论](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-fall-2011/)**  
  【核心优势】计算机顶校经典课，从复杂度分析讲到图算法，配套课后习题、实验和课件（可免费下载），理论深度拉满。  
  【适合人群】有 Python/C++ 基础，想系统学算法理论的人。

- **[Coursera：Stanford 算法专项课](https://www.coursera.org/specializations/algorithms)**  
  【核心优势】由算法大师 Tim Roughgarden 主讲，分4门课覆盖分治、贪心、动态规划、图算法，有中文字幕，结课可拿证书（部分企业认可）。  
  【适合人群】想进阶核心算法，需要“结构化学习路径”的开发者。

- **[慕课网：数据结构与算法入门](https://www.imooc.com/learn/1230)**  
  【核心优势】结合 Java 实现，侧重“工程实用”（如“链表在项目中的应用”），每节配实操作业，适合想边学边写生产级代码的人。


## 六、必备书籍【纸质书精读，打牢核心能力】
书籍适合“深度学习”，按“入门-进阶”分级，避免盲目啃大部头。

### 入门经典（新手优先读）
- **《算法图解》（原 Grokking Algorithms 中文版）**  
  【核心优势】全彩插图+生活化例子（如“用‘找钥匙’讲递归”），仅 200 多页，2周可读完，完全没有数学推导，新手入门首选。

- **《算法设计手册》**  
  【核心优势】分“理论篇”和“实战篇”：理论篇讲基础，实战篇收录 200+ 经典问题（附解题思路+代码），适合“学完基础就练题”的人。

- **《算法导论》**  
  【核心优势】业界“圣经”，覆盖算法理论全貌，但内容厚重（近 1300 页），建议有 1-2 年编程经验后再深入读，新手别直接啃。

### 进阶阅读（基础扎实后提升）
- **《编程之美》**  
  【核心优势】收集微软等大厂面试中的“工程算法题”（如“如何统计一篇文章的单词数”），侧重“算法落地能力”，适合想冲大厂的人。

- **《算法谜题》**  
  【核心优势】100 道谜题（如“约瑟夫环问题”），每题给多种解法分析，锻炼“一题多解”的思维灵活性，适合喜欢逻辑挑战的人。


## 七、实战练习平台【从练手到面试/竞赛】
算法的核心是“练”，按“目标”选平台，效率更高。

### 面试导向（冲大厂必备）
- **[LeetCode（国内版）](https://leetcode.cn/)**  
  【核心优势】中文界面+国内企业题库（阿里、腾讯等），有“热题100”“面试经典150”等专题，题解社区活跃（可看别人的优化思路）。  
  【使用建议】新手从“简单题”开始，每天1-2道，重点练“思路表达”（写注释/讲题）。

- **[CodeSignal](https://codesignal.com/)**  
  【核心优势】提供“算法能力评估测试”，生成可视化报告，部分企业（如谷歌、微软）用其筛选候选人，适合模拟面试场景。

### 竞赛导向（提升算法上限）
- **[Codeforces](http://codeforces.com/)**  
  【核心优势】全球顶级竞赛平台，分 Div.1（最难）到 Div.4（最简单），每周有比赛，适合想冲击竞赛奖项或提升“算法速度”的人。  
  【新手建议】从 Div.4 开始，先练“快速编码+bug 调试”。

- **[HackerRank](https://www.hackerrank.com/)**  
  【核心优势】按“技术方向”（算法、AI、后端等）和难度分类，部分题目有企业赞助，适合针对性提升某类技能。

### 趣味导向（边玩边练）
- **[Project Euler](https://projecteuler.net/)**  
  【核心优势】数学+算法结合的题目（如“计算第 1000 个质数”），前50题难度适中，适合锻炼“数学建模+算法实现”能力。

- **[Rosalind](https://rosalind.info/)**  
  【核心优势】生物信息学算法题库（如“DNA 序列分析”），边学算法边了解交叉学科，适合对生物/医学感兴趣的人。


## 八、辅助学习工具【提升练题/复盘效率】
这些工具帮你养成“规范解题习惯”，避免无效刷题。

- **[interactive-coding-challenges（GitHub）](https://github.com/donnemartin/interactive-coding-challenges)**  
  【核心优势】基于“测试驱动开发（TDD）”，每个挑战自带测试用例，支持 Python/Java 等语言，附带详细解题步骤，帮你养成“先想清楚再写代码”的习惯。

- **[AlgoExpert](https://www.algoexpert.io/)**  
  【核心优势】聚焦面试高频题，每道题有视频讲解+多种解法对比（暴力解→优化解），支持在线编程练习，适合短期冲刺大厂面试（部分内容付费）。


## 九、相关资源与学习路径【避坑指南】
### 1. 必备数学基础（算法的“底层逻辑”）
算法不需要高深数学，但这 3 类基础必须懂：  
- 离散数学（集合、图论）：对应图算法、树结构；  
- 概率论：对应随机算法、贪心策略；  
- 线性代数：对应 AI 相关算法（如动态规划优化）。  
推荐资源：《离散数学及其应用》（入门）、GitHub 仓库「awesome-math」（免费笔记）。

### 2. 科学学习路径（避免走弯路）
1. **入门（1-2个月）**：用 VisuAlgo 看可视化→读《算法图解》→刷 LeetCode 简单题（100道）；  
2. **基础（2-3个月）**：学 GeeksforGeeks 进阶内容→看 CodeSignal 复杂度分析→刷 LeetCode 中等题（150道）；  
3. **进阶（3-6个月）**：学 Coursera 专项课→读《算法设计手册》→冲 LeetCode 热题100/Codeforces Div.3；  
4. **冲刺（按需）**：刷目标企业题库（LeetCode 企业专区）→模拟面试（CodeSignal）。
